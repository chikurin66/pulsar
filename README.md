# PULSAR — Neon Survival

A neon **bullet-heaven** survivor (Vampire Survivors–like). Move only — your weapons auto-fire. Survive the swarm, level up, evolve your arsenal, and climb the daily leaderboard.

**▶ Play:** https://chikurin66.github.io/pulsar/

- **Controls:** WASD / arrow keys, or touch-drag on mobile. Attacks are automatic.
- Single-file HTML5 + Canvas2D. Works on desktop and mobile browsers.
- Daily seeded board (everyone gets the same run), streaks, shareable result cards.
- English / 日本語.
